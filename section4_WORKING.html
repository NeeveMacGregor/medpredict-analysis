üìä Current State: Manual Forecasting Process

Problem: Pharmacy managers currently use manual methods to forecast medication needs, leading to inaccurate predictions.

1Review Historical Usage
Pharmacy manager manually reviews past 30-60 days of medication usage in Excel spreadsheets
2Consider Seasonal Factors
Manager recalls seasonal patterns from memory (flu season, holidays, etc.)
3Check Current Inventory
Review current stock levels in Omnicell system
4Place Order
Manager estimates order quantities based on judgment and experience

Pain Points:

  • Time-consuming: 5-8 hours per week spent on manual analysis
  • Error-prone: Human judgment leads to over/understocking
  • Limited data: Only considers recent history, misses patterns
  • No predictive capability: Can't anticipate upcoming demand changes

‚úÖ Future State: MedPredict-Assisted Process

Solution: MedPredict automates forecasting using historical data, patient admissions, and seasonal trends.

1Automated Data Analysis
MedPredict analyzes Epic EHR admission data, Omnicell usage patterns, and seasonal trends automatically
2AI Recommendations
System generates medication demand forecasts with confidence levels
3Manager Review
Pharmacy manager reviews AI recommendations and adjusts if needed
4Streamlined Ordering
Manager approves orders based on system recommendations

Expected Benefits:

  • Time savings: Reduce forecasting time from 5-8 hours to 1-2 hours per week
  • Improved accuracy: Leverage historical patterns humans can't detect
  • Data-driven decisions: Replace gut feeling with evidence-based recommendations
  • Predictive capability: Anticipate demand spikes before they occur

üîÑ Key Process Changes

Staff Roles Affected

  • Pharmacy Managers (3 FTE): Shift from manual analysis to reviewing AI recommendations
  • Pharmacists: Receive better inventory information, fewer stockouts/overstocks
  • Pharmacy Technicians: Spend less time managing emergency orders

System Integration Points

  • Epic EHR: Pull patient admission data, diagnosis codes, treatment protocols
  • Omnicell: Import real-time medication usage and inventory levels
  • Purchasing System: Push order recommendations to procurement workflow

Training Requirements

  • Pharmacy managers: 4 hours on interpreting AI recommendations
  • Pharmacists: 2 hours on using new inventory dashboard
  • IT staff: 8 hours on system maintenance and troubleshooting
